به کمک این برنامه میتونید دیتا مورد نیاز برای کار روی داده های سایت TJGU رو بدون نیاز به عملیات دستی، از سایت دریافت کنید.
همچنین میتونید داده ها رو در یک فایل JSON برای برنامه نویسی و به عنوان ورودی داده به پلفرم های دیگه هم استفاده کنید.
تاریخ میلادی و شمسی، بالاترین و پایین ترین قیمت و قیمت در لحظه اغاز و پایان معاملات و میزان تغییرات قیمت به ریال و درصد، اطلاعاتی هستند که میتونید داشته باشید.
فقط کافیه به سایت مراحعه کنید و یک دارایی مانند دلار، یورو، سکه یا هر چیز دیگه رو انتخاب کنید و وارد صفحه مربوط به اون دارایی بشید، لینک صفحه رو کپی کنید و به برنامه بدید!
